# vim editor

A text editor which enables you to create and manipulate data in linux file

### Most common keys
```bash
i - insert

O - insert ont next line

a - insert after a space

Esc - Escape out of any mode

:q! - quit without saving

:wq! - quit and save

r - replace

d - delete

u - undo

x - remove one character

backspaace -
  
/ - forward search

? - backward search
```
